# Text Summarization Demo

This repository contains a demo application for text summarization using the Hugging Face Gradio SDK. The application is deployed to Hugging Face Spaces using GitHub Actions.

## Overview

The Text Summarization Demo uses a pre-trained text summarization model from the Hugging Face Transformers library. The application allows users to input text and receive a summarized version of the text.

## Features

- Input text for summarization.
- Receive a summarized version of the input text.
- Deployed to Hugging Face Spaces using GitHub Actions for continuous integration and deployment.
